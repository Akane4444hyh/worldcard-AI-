# world-card

A single-HTML flashcard app for vocabulary study, generated by a Python script.

## Features

- **Single HTML file** — no server, no dependencies. Open `flashcards.html` in any browser.
- **Dark / Light mode** — purple dark theme + pink-white light theme, persisted.
- **Group-based word management** — vertical group tabs, per-group counts and navigation.
- **Import** — supports `.json` (CET-6 format) and `.txt` files, or paste directly.
- **Slay & Restore** — mark words as known ("slay"), batch restore with drag-to-select.
- **Shuffle / Reset order** — alphabetical sort is the default; shuffle and restore anytime.
- **All data in localStorage** — no account, no server. Everything stays in your browser.

## Quick start

打开 `flashcards.html` 即可使用。双击文件用浏览器打开，或者命令行：

```bash
# macOS
open flashcards.html

# Windows
start flashcards.html

# Linux
xdg-open flashcards.html
```

如果需要从 Python 模板重新生成（修改了 `generate_flashcards.py` 之后）：

```bash
# macOS / Linux
python3 generate_flashcards.py

# Windows
python generate_flashcards.py
```

## Import formats

**TXT** — one word per line, separator can be ` — `, ` | `, tab, or space:
```
abandon — v. 放弃；遗弃
abuse — v. 滥用, n. 虐待
```

**JSON** — CET-6 format with `translations` and `phrases` arrays:
```json
[
  {
    "word": "abandon",
    "translations": [{"type": "v.", "translation": "放弃"}],
    "phrases": [{"phrase": "abandon oneself to", "translation": "沉溺于"}]
  }
]
```

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `←` `→` | Previous / Next card |
| `Space` / Click card | Flip |
| `K` | Slay (remove from study pool) |
| `R` | Reset to alphabetical order |
| `Escape` | Unflip card |

## Project structure

```
generate_flashcards.py   — Python template that generates flashcards.html
flashcards.html          — The app (can be used directly or regenerated from .py)
```
